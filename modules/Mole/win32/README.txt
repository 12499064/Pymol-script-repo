 How to
 ~~~~~~


 1) Install application from Mole.msi into directory "C:\Mole"

 2) Copy file "C:\Mole\mole.py" to your Pymol plugins directory 
    (usually c:\Program Files\DeLano Scientific\PyMOL\modules\pmg_tk\startup)

 3) See examples in C:\Mole\examples


 4) for more examples visit website: http://mole.chemi.muni.cz



 Enjoy New Mole!
